FAQ

분석 결과가 비어 있어요 — 표본 부족. "기관범위 → 자동확장" 또는 "직전 동일공종 우선" 토글 권장.

첫 분석 호출이 느려요 — 캐시 미적중 시 3~5초. 두 번째부터는 50ms 이내.

관리자 메뉴가 안 보여요 — 데모 계정은 read-only. 관리자 권한 필요.

특정 공고가 누락된 듯해요 — 취소/유찰/폐기/무효/단가계약은 자동 차단됨 (분석 정확도 보장).

데모 URL이 접속 안 돼요 — 안내 페이지에서 최신 URL 확인.

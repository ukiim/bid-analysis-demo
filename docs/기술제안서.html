01

핵심 기능

KBID 스타일 3단 분석 체계를 통해 1순위 예상 낙찰가를 도출하는 전문 플랫폼입니다. 용역 및 공사 항목만 취급합니다.

📋

공고화면

G2B/D2B 용역·공사 공고를 통합 수집하여 1순위 사정률 컬럼과 함께 표시합니다. 행 클릭 시 종합화면으로 이동하여 상세 분석을 시작합니다. 매일 자동 업데이트됩니다.

용역/공사 전용 1순위 사정률 일일 자동수집
📊

사정률 발생빈도 · 구간분석

동일 카테고리·기관 과거 데이터에서 0.01% 단위 빈도 히스토그램을 생성합니다. 최고빈도 구간 자동 식별, 기간별(3/6/9/12/24개월) · 기관별(개별/상위) 범위 전환을 지원합니다.

빈도 히스토그램 클릭 인터랙션 밀어내기 기간분석
🔬

업체사정률 분석 (갭 분석)

선택 구간 내 과거 업체 투찰률을 정렬하고 연속 값 간 최대 빈 공간(갭)을 탐색합니다. 갭 중간점이 정제된 사정률(확정사정률)로 도출되며, 1순위 예측 후보군을 제공합니다.

갭 중간점 도출 투찰률 분포 1순위 후보군
🏆

종합분석 · 1순위 예상 낙찰가

확정사정률을 과거 데이터에 적용하여 1순위 매칭을 검증합니다. 3/6/9/12/24개월 기간별 적중률을 동시 표시하고, 발주처별 상위범주(시→도) 분석을 수행합니다. 최종 출력: 1순위 예상 낙찰가

기간별 적중률 상위기관 분석 최종 낙찰가 출력
🔐

사용자 인증 · 권한 관리

JWT 기반 회원가입/로그인 시스템을 통해 사용자를 인증합니다. 역할 기반 접근 제어(user/admin)로 관리자 기능을 분리하고, 분석 조회 이력을 사용자별로 추적합니다.

JWT 인증 역할 기반 접근 조회 이력 추적
📤

데이터 업로드 · 관리

CSV/Excel 파일을 통해 입찰 데이터를 직접 업로드할 수 있습니다. 필수 컬럼 검증 및 자동 DB 적재를 지원하며, 업로드 이력을 관리합니다.

CSV/Excel 지원 데이터 검증 업로드 이력

1순위 예상 낙찰가 핵심 알고리즘

KBID 스타일 3단 분석 프로세스를 통해 최적 사정률을 도출하고, 최종 1순위 예상 낙찰가를 산출합니다.

3단 분석 상세

단계분석 방법입력출력
Step 1. 사정률 발생빈도 동일 카테고리·지역·기관 과거 낙찰결과에서 0.01% 단위 빈도 히스토그램 생성 기간(3/6/9/12/24개월), 기관범위(개별/상위) 빈도 분포, 피크 구간, 통계(평균/중앙/표준편차)
Step 2. 선택사정률 결정 히스토그램에서 사용자가 최고빈도 구간을 클릭하여 선택 빈도 히스토그램 클릭 이벤트 선택사정률 (예: 99.3%)
Step 3. 업체 갭 분석 선택구간(±0.5%) 내 업체 투찰률 정렬 후 연속 값 간 최대 빈 공간 탐색, 갭 중간점 산출 선택사정률, 기간 정제된 사정률(확정사정률), 갭 크기·위치, 1순위 후보군
Step 4. 과거 검증 확정사정률을 과거 1년 데이터에 적용, 실제 1순위와 비교하여 적중률 산출 확정사정률, 기간별(3~24개월), 기관범위 기간별 매칭률, 상위기관 분석, 비교 테이블
Step 5. 최종 산출 1순위 예상 낙찰가 = 기초금액 × 확정사정률 / 100 기초금액, 확정사정률 1순위 예상 낙찰가(원), 예측 사정률

NAS 스토리지 (2TB)

모든 수집 데이터와 분석 결과를 NAS 서버에 저장합니다. API 원본 응답(JSON), SQLite DB 파일, 모델 파일, 내보내기 데이터를 체계적으로 관리합니다.

데이터 송수신 체계

공공데이터 수집 및 3단 분석 API
  • G2B/D2B API 클라이언트 (용역/공사 전용 필터)

Phase 3 26~45일 (20일) — Frontend 개발
공고화면 + 종합화면 3탭 구현

## 2. 다음 주 작업 계획 (2주차: 2026-04-27 ~ 2026-05-01)

### 2-1. 공공데이터 API 연동
- 나라장터(G2B) 입찰공고 조회 API 연동
- 개찰결과 조회 API 연동
- API 호출 오류 처리 및 재시도 로직 구현
- Rate Limit 대응 큐/스케줄러 설계

### 2-2. 데이터 자동 수집 배치
- APScheduler 기반 자동 수집 배치 구현 (매시/매일/매주 옵션)
- 수집 결과 `SyncHistory` 자동 기록
- 수집 실패 시 관리자 알림(이메일) 연동

### 2-3. 분석 엔진 1차 구현
- **Tab1 빈도 분석**: 사정률 빈도 분포 계산, 1순위 예측 후보 리스트, 과거 1순위 비교 테이블
- **Tab2 갭 분석**: 사정률 간 갭 계산, 차기연도 검증 테이블
- **Tab3 종합 분석**: Tab1·Tab2 결합 예측, 밀어내기식 검토, 연도별 검증

### 2-4. NAS 환경 준비 (클라이언트 협업)
- NAS 모델·OS·Docker 지원 여부 확인 (클라이언트 회신 대기)
- Docker Compose 초안 작성 (backend / frontend / nginx / redis)
- Cloudflare Tunnel 연동 가이드 작성

### 2-5. 테스트 및 QA
- 주요 API 엔드포인트 pytest 단위 테스트 작성
- 프론트엔드 E2E 시나리오 초안 (Playwright)
- 1차 검수(2026-05-20) 대비 체크리스트 준비

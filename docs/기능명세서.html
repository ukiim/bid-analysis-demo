3 사정률 발생빈도 분석
과거 낙찰 데이터의 사정률 분포를 히스토그램으로 분석하고, 1순위 예측 후보를 산출합니다.
FREQ-01 사정률 히스토그램 + 1순위 예측 후보 필수
화면종합화면 → Tab1 "사정률 발생빈도와 구간분석"
기능 설명 선택 공고와 동일 카테고리/발주처의 과거 사정률을 0.1% 단위 히스토그램으로 시각화.
기준일: 공고일(announced_at) 이전 데이터만 사용.
각 구간(bin)에 사정률 빈도 + 1순위 횟수를 함께 집계.
반환 데이터 bins 히스토그램 데이터 (rate, count, first_place_count)
peaks 상위 5개 피크 구간
stats 평균, 중앙값, 표준편차, 최소, 최대
prediction_candidates 1순위 예측 후보 15개 (점수순)
예측 점수 score = freq_score * 0.5 + fp_score * 0.5
freq_score: 빈도 비율 (해당 구간 빈도 / 최대 빈도)
fp_score: 1순위 비율 (해당 구간 1순위 횟수 / 최대 1순위 횟수)
UI 구성 1. 기간/기관 컨트롤바
2. 막대 차트 (클릭 → 사정률 선택)
3. 통계 카드 3개
4. 1순위 예측 후보 테이블 (순위, 사정률, 빈도, 1순위 횟수/비율, 점수바, 투찰금액, ★추천, 선택 버튼)
GET /api/v1/analysis/frequency/{announcement_id}

org_scopestringspecific|parentspecific기관 범위
4 업체사정률 갭 분석
선택 구간 내 업체 투찰률 분포에서 빈 공간(갭)을 찾아 최적 사정률을 추천합니다.
GAP-01 업체 투찰률 분포 + 갭 분석 필수
화면종합화면 → Tab2 "업체사정률 분석"
전제 조건Tab1에서 선택사정률이 설정되어 있어야 함
기능 설명 선택사정률 ±0.5% 범위 내 업체 투찰률을 조회하고, 연속 투찰률 간 최대 빈 공간(갭)을 식별.
갭 중간점을 확정사정률 후보로 추천.
갭 산출 유의미한 갭: 0.01% 초과
최대 10개 갭을 크기순 정렬
각 갭: start, end, size, midpoint
반환 데이터 company_rates 업체 투찰률 (최대 100건)
gaps 갭 목록 (최대 10개)
first_place_predictions 해당 구간 1순위 실적
next_year_validation 차기연도 검증 결과
GET /api/v1/analysis/company-rates/{announcement_id}
파라미터타입기본값설명
rate_range_startfloat99.0분석 시작 사정률
rate_range_endfloat100.0분석 종료 사정률
period_monthsint12분석 기간
5 종합분석
확정사정률로 과거 데이터를 기간별로 백테스팅하여 1순위 적중률을 검증합니다.
COMP-01 기간별 매칭률 검증 필수
화면종합화면 → Tab3 "종합분석"
전제 조건확정사정률이 설정되어 있어야 함
기능 설명 확정사정률로 과거 1, 3, 6, 12, 24개월 데이터에서 1순위 달성 여부를 검증.
매칭 기준: |확정사정률 - 실제사정률| ≤ |1순위사정률 - 실제사정률| + 0.02%
UI 구성 1. 1순위 예상 낙찰가 카드
2. 투찰 전략 요약 (투찰금액, 권장범위, 적중확률)
3. 기간별 매칭률 바 차트
4. 상위기관 분석 패널
5. 과거 비교 테이블
GET /api/v1/analysis/comprehensive/{announcement_id}
파라미터타입기본값설명
confirmed_ratefloat99.5확정사정률

6 결합 예측 (빈도 + 갭)
빈도분석 피크와 갭분석 중간점을 결합하여 최적 후보 10개 이상을 산출합니다.
COMB-01 결합 예측 후보 산출 필수
화면종합화면 → Tab3 내 "결합 예측 후보" 테이블
기능 설명 1단계: 빈도 히스토그램 상위 10개 피크 구간 추출
2단계: 업체 투찰률 갭 중간점 상위 15개 추출
3단계: 피크 + 갭 결합 점수 산출
점수 공식 combined_score = freq_score * 0.4 + gap_score * 0.4 + fp_score * 0.2
- freq_score: 해당 구간 빈도 / 최대 빈도
- gap_score: 해당 갭 크기 / 최대 갭 크기
- fp_score: 해당 구간 1순위 횟수 / 최대 1순위 횟수
출처 분류 빈도+갭 양쪽 모두 해당
빈도 빈도 피크에서만 발견
갭 갭 중간점에서만 발견
신뢰도 높음: score > 0.6  |  보통: score > 0.3  |  낮음: score ≤ 0.3
테이블 컬럼순위, 사정률, 출처, 빈도, 1순위, 빈도점수, 갭점수, 결합점수, 예상투찰금액, 신뢰도, 확정 버튼
GET /api/v1/analysis/combined-prediction/{announcement_id}
7 밀어내기식 검토 (Sliding Window)
시간 윈도우를 이동하며 확정사정률의 1순위 적중 여부를 시계열로 검증합니다.
SLIDE-01 슬라이딩 윈도우 백테스팅 필수
화면종합화면 → Tab3 내 "밀어내기식 검토" 섹션
기능 설명 윈도우 크기만큼의 과거 데이터를 순차적으로 이동하면서, 각 시점에서:
1. 윈도우 내 사정률 중앙값으로 예측
2. 다음 시점의 실제 결과와 비교
3. 확정사정률의 1순위 달성 여부 판정
윈도우 크기 1일 5일 1개월 3개월 6개월 1년
시각화 타임라인 라인 차트 (3개 선):
- 실제 사정률 (파란색, 실선)
- 1순위 사정률 (녹색, 점선)
- 확정사정률 (빨간색, 점선)
+ 적중률 통계 카드
GET /api/v1/analysis/sliding-review/{announcement_id}
파라미터타입기본값설명
window_sizestring3m1d, 5d, 1m, 3m, 6m, 1y
confirmed_ratefloat99.5확정사정률
org_scopestringspecific기관 범위
8 연도별 낙찰확률 검증
2020~2026년 각 연도별로 확정사정률의 1순위 적중률을 검증합니다.
YEAR-01 연도별 적중률 검증 필수
화면종합화면 → Tab3 내 "연도별 낙찰확률 검증" 섹션
기능 설명 각 연도(2020~2026)에서 확정사정률로 1순위가 되었을 건수 / 전체 건수를 계산.
막대 차트 + 연도별 펼치기(상세 케이스 테이블) 제공.
색상 기준 60% 이상 녹색  |  30~60% 노란색  |  30% 미만 빨간색
상세 테이블공고명, 발주기관, 실제사정률, 1순위사정률, 날짜, 적중 여부
GET /api/v1/analysis/yearly-validation/{announcement_id}
9 추첨예가 빈도 분석
복수예비가격 15개 번호 중 추첨되는 빈도를 분석합니다.
PRELIM-01 추첨번호 빈도 히스토그램 보통
화면종합화면 → Tab1 내 "추첨된 예가 빈도수" 차트
기능 설명 복수예비가격 15개 번호(#1~#15) 중 매회 4개가 추첨되는데, 과거 데이터에서 각 번호의 추첨 빈도를 막대 차트로 시각화.
높은 빈도 번호는 노란색 하이라이트.
폴백동일 발주기관 데이터 부족 시 → 동일 카테고리+지역으로 확장 조회
GET /api/v1/analysis/preliminary-frequency/{announcement_id}
10 갭 차기연도 검증
갭 분석에서 식별된 중간점이 다음 연도 실제 결과와 얼마나 일치하는지 검증합니다.
GAPV-01 갭 중간점 차기연도 적중률 보통
화면종합화면 → Tab2 하단 "갭 차기연도 검증" 테이블
기능 설명 공고일 이후 1년간의 실제 낙찰 데이터에서, 각 갭 중간점의 1순위 적중 여부를 검증.
상위 10개 갭에 대해 적중률 산출.
판정 기준 유효 적중률 ≥ 50%  |  보통 25~50%  |  미흡 < 25%
테이블 컬럼갭 중간점, 갭 크기, 검증 건수, 적중 건수, 적중률(바), 결과 판정
